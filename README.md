## 프로젝트 구조
```
Machine Learning/
 ├─ .git/          
 ├─ .gitignore
 ├─ venvs/          ← 가상환경 (깃 무시)
 ├─ notebooks/      ← 주피터 노트북 저장할 폴더
 │   ├─ a.ipynb
 │   └─ b.ipynb
 ├─ src/            ← 파이썬 코드 (x)
 └─ requirements.txt
```
- python3 -m venv ML 로 가상환경 생성
- 가상환경 활성화
   - source venvs/ML/bin/activate
